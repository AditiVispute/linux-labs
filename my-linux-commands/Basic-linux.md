# Basic Linux Commands

## 1. pwd
Print current working directory.

pwd

---

## 2. ls
List files and directories.

ls
ls -l
ls -a

---

## 3. cd
Change directory.

cd /home
cd ..
cd ~

---

## 4. mkdir
Create a new directory.

mkdir testdir

---

## 5. rmdir
Remove empty directory.

rmdir testdir

---

## 6. touch
Create empty file.

touch file.txt

---

## 7. cp
Copy files and directories.

cp file1.txt file2.txt
cp -r folder1 folder2

---

## 8. mv
Move or rename files.

mv file1.txt newfile.txt
mv file.txt /home/user/

---

## 9. rm
Delete files and directories.

rm file.txt
rm -r foldername

---

## 10. clear
Clear terminal screen.

clear

---

## 11. echo
Print text to terminal.

echo "Hello Linux"

---

## 12. cat
Display file content.

cat file.txt

---

## 13. nano
Open file in editor.

nano file.txt

---

## 14. history
Show previously used commands.

history

---

# Practical Lab

1. Create directory:
   mkdir practice

2. Go inside:
   cd practice

3. Create file:
   touch test.txt

4. Write content:
   echo "Hello World" > test.txt

5. View file:
   cat test.txt

6. Copy file:
   cp test.txt copy.txt

7. Rename file:
   mv copy.txt new.txt

8. Delete file:
   rm new.txt
